# Spec Gap Audit — Index

本資料夾收錄歷次 Pantheon Frontend Build Spec 與現有實作之間的「規格缺漏 / 矛盾」盤點報告。每份報告獨立存檔，命名規則：

```
spec-gap-YYYY-MM-DD-{流水序 A/B/C…}.md         # 完整報告（zh-TW）
spec-gap-YYYY-MM-DD-{流水序}-summary.csv       # 試算表摘要
```

| 版本 ID | 日期 | 範圍 | 條目數 | 高 / 中 / 低 | 對應實作 commit | 備註 |
|---------|------|------|--------|--------------|-----------------|------|
| `spec-gap-2026-05-05-A` | 2026-05-05 | Part 1–8 全量盤點 | 92 | 27 / 41 / 24 | 當前 main | 首次完整盤點；提供給規劃團隊作為 spec 反饋輸入 |

## 使用建議

1. **規劃團隊**：以 `-summary.csv` 篩選 severity=High，逐條補定義；補完後在主報告對應條目標記 `RESOLVED`。
2. **後續審計**：請新增列，不覆蓋既有版本；命名請保留日期 + 字母流水。
3. **與 plan.md 關係**：本資料夾不修改 `src/` 程式；若決議導致實作異動，由規劃團隊另開 plan 後再執行。
